``GitHub`` installation
------------------------

To use the modified version of ``lenstool``, first ``git clone`` it:

.. code-block:: console

   git clone https://github.com/njzifjoiez/Lenstool_Bspline

and switch branch:

.. code-block:: console

   git checkout idPIE-pot-joint
